# OOPS Lab Assignments

This repository contains C++ programs for OOPS lab assignments.

## Assessment 5: Inheritance

The `assessment-5` folder contains the programs for the inheritance assessment.

| File | Topic |
| --- | --- |
| `q1_simple_inheritance.cpp` | Simple base and derived class example |
| `q2_protected_member.cpp` | Protected member access in inheritance |
| `q3_inheritance_modes.cpp` | Public, protected, and private inheritance modes |
| `q4_types_of_inheritance.cpp` | Single, multiple, hierarchical, multilevel, and hybrid inheritance |
| `q5_constructor_destructor.cpp` | Constructors and destructors in inheritance |
| `q6_book_textbook.cpp` | Book and Textbook single inheritance example |

## Compile and Run

Open a terminal inside the repository and compile any program separately.

```bash
g++ assessment-5/q1_simple_inheritance.cpp -o q1
./q1
```

## Assessment 6

The `assessment-6` folder contains programs for array of objects, constructors, pointer to objects, passing and returning objects, static members, const member functions, inline functions, and friend functions.

| File | Topic |
| --- | --- |
| `q1_library_array.cpp` | Library class with array of Book objects |
| `q2_library_constructors.cpp` | Library program using constructors and this pointer |
| `q3_account_transactions.cpp` | Account transactions using const data and const member function |
| `q4_friend_function.cpp` | Adding objects of two different classes using a friend function |
| `q5_complex_friend.cpp` | Complex number addition using a friend function |
